# reactjs-2019q1

## Init step
* `git clone https://github.com/var-bin/reactjs-2019q1.git`
* `yarn`

## Dev Env
* `yarn start`

## Prod Env
* `yarn build`

## Test Env
* `yarn test`
* `yarn test:watch // watch mode`
* `yarn test:coverage // add coverage`
* `yarn e2e // e2e tests via cypress`
* `yarn e2e:open // e2e tests via cypress and open browser`
